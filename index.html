Displ

Облачная платформа для управления информационными дисплеями: профессиональные панели Samsung со встроенным проигрывателем или любые другие дисплеи с использованием портативного внешнего проигрывателя. Управление осуществляется через единый веб-интерфейс.

Основные сферы использования: меню борды для ресторанов быстрого питания; рекламные панели для ритейла; информационные табло для банков, аэропортов, вокзалов, кинотеатров, торговых центров, отелей; рекламные панели в такси; корпоративное телевидение.

Наши возможности

Добавление дисплеев

Добавьте все ваши дисплеи в единую облачную платформу.

Разбиение на группы

Разделите ваши дисплеи на разные группы по какому-либо признаку.

Загрузка контента в облако

Загрузите весь ваш контент (фото, видео, документы) на сервер.


Создание шаблонов проигрывания

Создайте шаблон с помощью загруженного контента.

Интеграция с внешними данными

Отображайте поверх шаблона данные из внешних источников и баз данных.

Оффлайн проигрывание шаблонов

Дисплеи не требуют постоянного подключения к интернету. Контент предзакачивается во внутреннюю память.


Создание расписаний шаблонов

Задавайте правила показа шаблонов на экранах.

Привязка расписания к группам

Привяжите расписание к определенной группе дисплеев.

Мониторинг состояния устройств

Контролируйте какие дисплеи на связи, а какие нет и смотрите что сейчас транслируется на каждом из них.
